HOW TO USE

  • Hover over 'a user' to view author name.
  • Press 'download' to download a copy of the message.
  • You can generate as many prompts as you want using the 'generate' button on the top left.
  • Leaving the website will delete the previously generated prompts. This includes going to the help page.
